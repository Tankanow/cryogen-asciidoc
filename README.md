# cryogen-asciidoc

A Clojure library to provide AsciiDoc rendering to the cryogen-core compiler

## Usage

Add `[cryogen-asciidoc "0.1.1"]` to the `:dependencies` keyword of the `project.clj` of your generated cryogen blog.
